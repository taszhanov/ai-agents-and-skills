# ai-agents-and-skills
Готовые конфигурации скиллов и суб-агентов которые вы можете переиспользовать для вашего ИИ-агента (Claude, Gemini CLI, etc.).

УСТАНОВКА АГЕНТОВ И СКИЛЛОВ ДЛЯ CLAUDE CODE
Шаг 1. Скачайте файлы SKILLS.md и AGENTS.md.
Шаг 2. Откройте папку .claude на вашем компьютере.
Шаг 3. В левой панели найдите папку skills - она есть по умолчанию. Положите туда файлы скиллов.
Шаг 4. В левой панели найдите папку agents - она есть по умолчанию. Положите туда файлы агентов.
Шаг 5. Сохраните изменения.
Шаг 6. Перезагрузите Claude Code.
Шаг 7. Готово! Все скиллы и субагенты теперь доступны.
