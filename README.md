## 📦 Требования

- Claude Code установлен на компьютере
- Или: Gemini CLI установлен на компьютере
- Текстовый редактор для редактирования файлов

## 🚀 Быстрый старт

### Для Claude Code

1. Скачайте репозиторий или файлы `SKILLS.md` и `AGENTS.md`
2. Откройте папку `.claude` на вашем компьютере
3. Скопируйте `SKILLS.md` в папку `skills/`
4. Скопируйте `AGENTS.md` в папку `agents/`
5. Перезагрузите Claude Code
6. Готово! ✅

## ⚙️ Установка для Claude Code

### Шаг 1. Скачайте файлы

Скачайте файлы `SKILLS.md` и `AGENTS.md` из этого репозитория.

### Шаг 2. Откройте папку .claude

На вашем компьютере найдите скрытую папку `.claude`:

**Windows:**
```
C:\Users\[ВашИмя]\.claude
```

**macOS / Linux:**
```
~/.claude
```

### Шаг 3. Разместите скиллы

Положите файл `SKILLS.md` в папку:
```
.claude/skills/SKILLS.md
```

### Шаг 4. Разместите агентов

Положите файл `AGENTS.md` в папку:
```
.claude/agents/AGENTS.md
```

### Шаг 5. Сохраните изменения

Убедитесь, что файлы сохранены. Используйте клавиши `Ctrl+S` (Windows/Linux) или `Cmd+S` (macOS).

### Шаг 6. Перезагрузите Claude Code

Закройте и откройте Claude Code заново, или используйте команду перезагрузки в интерфейсе.

### Шаг 7. Готово! ✅

Все скиллы и суб-агенты теперь доступны в вашем Claude Code.

## ⚙️ Установка для Gemini CLI

### Шаг 1. Создайте папку для конфигураций

```bash
mkdir -p ~/gemini-agents
```

### Шаг 2. Скопируйте файлы

```bash
cp SKILLS.md ~/gemini-agents/
cp AGENTS.md ~/gemini-agents/
```

### Шаг 3. Создайте конфиг-файл

В вашем проекте создайте файл `.env`:

```env
AGENTS_CONFIG_PATH=~/gemini-agents
SKILLS_FILE=SKILLS.md
AGENTS_FILE=AGENTS.md
```

### Шаг 4. Готово! ✅

Все скиллы и агенты теперь загружаются автоматически.
